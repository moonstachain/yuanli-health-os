# Yuanli Health OS Strategy

## Position

Yuanli Health OS is a public desktop demo and product shell for an AI-native personal health operating system.

## Roadmap

### v1 Public site

Show the product concept, visual style, and desktop demo.

### v2 Local data layer

Use local or private data files for personal health records.

### v3 Agent layer

Generate daily brief, Zone2 plan, sleep plan, weekly review, and missing data queue.

### v4 Recursive experiments

Run 14-day experiments and quarterly review loops.

## Public repo boundary

The public repo should contain demo data, UI, docs, schema notes, and Codex tasks. Personal reports and private health exports should stay local or private.
